WHAT THIS DOES CADS V3.0 is a Dynamic Voltage and Frequency Scaling (DVFS) simulator. It calculates the minimum required frequency to meet a task's deadline, significantly reducing power consumption ($P \propto V^2f$). This version specifically monitors core behavior to detect Power Viruses (tasks that intentionally drain battery or cause thermal stress) and includes Leakage Mitigation logic to decide when running faster and entering deep sleep is more efficient than running slowly.
HOW TO USE IT 1. Configure Tasks: Set workload cycles and deadlines.
2. Priority: Use 'Critical' for immediate processing or 'Suspect' to test the Security Sentinel.
3. Execution: Hit 'Start Kernel'. Observe how Core 0 handles tasks. If a task is too heavy, the system flags it in red.
4. Analyze: Watch the 'Energy Saved' metric. The higher the slack (time available), the more juice you save.
